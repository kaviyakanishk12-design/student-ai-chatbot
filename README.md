# student-ai-chatbot
AI Chatbot for Student Queries is a simple web-based chatbot that answers common academic questions such as courses, exams, placements, and internships. Built using HTML, CSS, and JavaScript, it provides an interactive chat interface and rule-based responses to help students quickly access important information.
Project Overview

AI Chatbot for Student Queries is a simple web-based chatbot that helps students get instant answers to common academic questions. The chatbot can respond to queries related to courses, exams, placements, and internships. It provides a user-friendly chat interface and uses rule-based logic to generate responses.

🚀 Features

1.Interactive chatbot interface

2.Instant responses to student queries

3.Simple and clean UI design

4.5.Handles common questions about courses, exams, placements, and internships
